Berbeiten

In diesem Abschnitt werden die Funktionen des Menüs "Bearbeiten" erklärt.

Rückgängig

Eingaben können mit diesem Befehl rückgängig gemacht werden. Die Funktion kann auch über die Tastenkombination Strg + Z aufgerufen werden.

Wiederholen

Zuvor rückgängig gemachte Eingaben können mit diesem Befehl wiederhergestellt werden. Die Funktion kann auch über die Tastenkombination Strg + Y aufgerufen werden.

Ausschneiden

Dieser Befehl dient zum Ausschneiden markierter Textpassagen. Der Text landet in der Zwischenablage und kann später wieder über die Funktion "Einfügen" an beliebiger Stelle eingefügt werden. Die Funktion kann auch über die Tastenkombination Strg + X aufgerufen werden.

Kopieren

Dieser Befehl dient zum Kopieren markierter Textpassagen. Der Text landet in der Zwischenablage und kann später wieder über die Funktion "Einfügen" an beliebiger Stelle eingefügt werden. Die Funktion kann auch über die Tastenkombination Strg + C aufgerufen werden.

Einfügen

Dieser Befehl fügt eine zuvor kopierte oder ausgeschnittene Textpassage wieder an beliebiger Stelle ein. Die Funktion kann auch über die Tastenkombination Strg + V aufgerufen werden.

Löschen

Dieser Befehl löscht eine zuvor markierte Textpassage aus dem Eingabefeld. Diese Funktion löscht lediglich den markierten Text. Der Inhalt der Zwischenablage wird durch die Ausführung des Kommandos nicht verändert. Die Funktion kann auch über die Taste Entf aufgerufen werden.

Alles markieren

Mit diesem Befehl können Sie eine Auswahl erstellen, welche den kompletten Text im Eingabefeld umfasst. Die Funktion kann auch über die Tastenkombination Strg + A aufgerufen werden.
