AssemblerSim

Die Doku zum Programm

Datei

Das Menü "Datei" stellt zentrale Funktionen zur Arbeit mit AssemblerSim zur Verfügung.

Neu

Ein neues Dokument wird eröffnet und bestehende Eingaben werden verworfen. Die Funktion kann auch über die Tastenkombination Strg + N aufgerufen werden.

Öffnen

AssemblerSim beherrscht momentan ausschließlich Dateien mit der Endung "asm" und "a51". Diese können über das Öffnen-Menü geladen werden. Die Funktion kann auch über die Tastenkombination Strg + O aufgerufen werden.

Speichern

Um den geschriebenen Code zu sichern, kann man ihn über die Funktion "Speichern" ins "asm"-Format exportieren. Die Funktion kann auch über die Tastenkombination Strg + S aufgerufen werden.

Neustarten

Unterbricht alle laufenden Aktivitäten des Simulators und startet AssemblerSim neu.

Beenden

Unterbricht alle laufenden Aktivitäten des Simulators und beendet das Programm. Die Funktion kann auch über die Tastenkombination Strg + Q aufgerufen werden.

Achtung: Ungespeicherte Daten können nach dem Beenden nicht mehr wiederhergestellt werden!
