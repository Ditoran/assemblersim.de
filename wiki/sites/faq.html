FAQ

Wird für den Betrieb zwingend Java 7 benötigt?

Ja, um AssemblerSim betreiben zu können, wird Java 7 benötigt. Für den Fall, dass Sie Java nicht installieren möchten, keine Updates der bestehenden Version durchführen können oder keine Berechtigung haben Java zu installieren, können Sie auf die portable Version zurückgreifen. Diese enthält Java 7, welches ohne Installation direkt startbereit ist.

Ist der Quellcode öffentlich zugänglich?

Momentan ist noch nicht absehbar, ob und wann der Quellcode veröffentlicht wird. Wenn Sie aber Fragen zum Programm bzw. zur Funktionalität eines Elements haben, können Sie sich gerne per Kontaktformular melden. Quellcode direkt wird zwar nicht verschickt, aber Tipps zur Umsetzung erhalten Sie auf jeden Fall.
